# SC-008A Design Brief

## Goal

Package the current ChatBridge proof into one reviewer-facing entry doc so the
grader can find setup, deployment, architecture, seeded prod sessions, and the
graded-scenario checklist without reverse engineering the repo.

## Scope

- One canonical submission index under `chatbridge/`
- Explicit mapping from each graded scenario to real tests and, when available,
  seeded prod sessions
- Minimal linking updates to the existing ChatBridge brief and deployment docs

## Non-Goals

- No new product UI
- No partner quickstart rewrite yet
- No AI cost analysis yet

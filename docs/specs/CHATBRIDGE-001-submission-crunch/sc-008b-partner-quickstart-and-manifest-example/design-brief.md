# SC-008B Design Brief

## Goal

Turn the existing partner contract reference into a real onboarding surface for
new reviewed-app developers.

## Scope

- extend `chatbridge/PARTNER_SDK.md` into a true quickstart
- add one checked-in reviewed manifest example artifact
- make the reviewer packet point to the quickstart and example
- add a contract test so the docs stay aligned with the validator and harness

## Non-Goals

- no runtime code changes
- no new hosted UI
- no cost-analysis work
